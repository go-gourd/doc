下面演示如何快速创建一个Gourd项目

## 拉取项目代码

> 使用git下载

```bash
git clone https://github.com/go-gourd/gourd-app.git
```

> 前往Github下载  

[https://github.com/go-gourd/gourd-app](https://github.com/go-gourd/gourd-app)

## 运行Demo

下载依赖
```bash
go mod tidy
```

开始运行
```bash
go run main.go
```

等待控制台输出
