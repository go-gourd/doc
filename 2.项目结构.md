### 内置事件如下：
> Boot 事件：框架初始化前执行

> Init 事件：框架初始化完成后执行

> Start 事件：框架启动后执行

> Stop 事件：进程关闭前执行 

> HttpStart 事件：Http服务启动后执行

### 如何注册事件回调？
在`internal/event/app.go`中`Register`方法内定义事件注册

系统内置事件都以下划线_开头,驼峰也需使用下划线

如Start事件：_start，HttpStart 事件：_http_start

```go
// Boot事件(框架) -启动时执行
event.AddEvent("_boot", func(params any) {
    logger.Debug("boot event.")
})
```
