
本项目使用被广泛应用的结构，细节可参考：[golang-standards/project-layout](https://github.com/golang-standards/project-layout)

```

├─config     配置文件目录
│  ├─app.toml    应用配置文件
│  ├─log.toml    日志配置文件
│  └─...
├─internal   应用目录
│  ├─app
│  │  ├─api      api应用（演示）
│  │  │  ├─ctl      控制器目录
│  │  │  │  ├─user.go   用户控制器
│  │  │  │  └─...
│  │  │  ├─route    路由定义
│  │  │  │  └─route.go  用户模型
│  │  │  └─...
│  │  └─...
│  ├─cmd      命令行
│  │  └─cmd.go  命令行注册
│  ├─cron     定时任务
│  │  └─cron.go  定时任务注册
│  ├─event    事件
│  │  └─app.go  事件注册
│  ├─orm    ORM代码
│  │  ├─model  模型
│  │  └─query  查询
│  ├─router  HTTP路由服务
│  │  ├─router.go  路由注册
│  │  └─server.go  HTTP服务
│  └─...
├─runtime   运行时目录
│  ├─log      日志目录
│  └─...
├─scripts    脚本目录
│  ├─build.bat    编译脚本
│  ├─build.sh     编译脚本
│  ├─build.ps1    编译脚本
│  └─...
├─test       测试目录
├─web        Web目录
│  ├─static    静态资源目录
│  └─...
├─.gitignore  git忽略文件
├─go.mod      go模块文件
├─LICENSE     许可证文件
├─README.md   项目说明文件
```
